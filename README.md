# Speech_Emotion-Semetic_Classification
The Given Notebook explores MFCC, Chroma, Mel spectrogram, and zero-crossing rate for classification of emotion of voice.
The Voice Classes are
 '01': 'neutral',
 '02': 'calm',
 '03': 'happy',
 '04': 'sad',
 '05': 'angry',
 '06': 'fearful',
 '07': 'disgust',
 '08': 'surprised
